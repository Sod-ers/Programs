### Install on Linux:
1. Install [Bottles](https://flathub.org/en/apps/com.usebottles.bottles).
2. Create new bottle, default settings.
3. Run executable from Downloads (TouchDesigner web setup).
4. Install `allfonts` dependency (Options section in this menu).
5. Run program or add shortcut if not automatically added.
6. (Optional) Run Terminal command `flatpak override com.usebottles.bottles --user --filesystem=xdg-data/applications` to give Bottles permission to add desktop icon for TouchDesigner. (Located at hamburger menu of TouchDesigner program section).
  
Setup menu is laggy but program itself runs great.
